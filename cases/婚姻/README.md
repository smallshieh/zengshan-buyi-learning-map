## 💡 核心學習要點

### 1. 用神判斷

#### 男占婚姻
- **用神**：妻財爻（代表女方）
- **原神**：子孫爻（生財）
- **忌神**：兄弟爻（克財）

#### 女占婚姻
- **用神**：官鬼爻（代表男方）
- **原神**：財爻（生官）
- **忌神**：子孫爻（克官）

### 2. 成婚判斷

✅ **成婚之象**：
- 財爻持世旺相（男占）
- 官鬼持世旺相（女占）
- 財化進神
- 官化進神
- 世應相生相合
- 六合卦

❌ **不成之象**：
- 財化退神（男占）
- 官化退神（女占）
- 世爻空破
- 用神空破
- 兄弟持世動（男占）
- 子孫持世動（女占）
- 六沖卦

### 3. 阻礙判斷

#### 有阻礙
- 間爻官鬼動（有人阻撓）
- 兄弟動克財（男占，兄弟爭奪）
- 子孫動克官（女占，長輩反對）

#### 阻礙可解
- 鬼化退神（雖阻無力）
- 兄化退神（阻力減弱）
- 逢合解沖（化解矛盾）

### 4. 應期判斷

#### 成婚應期
```
財化進神 → 進神得令之日
官化進神 → 進神得令之日
世應相合 → 合之日
用神出空 → 出空之日
```

#### 允婚應期
```
回頭克解除 → 沖去克神之日
財爻逢合 → 合之日
官爻逢合 → 合之日
```

## 🎯 實戰判斷流程

### 男占婚姻

```
1. 看財爻
   ├─ 持世旺相 → 可成
   ├─ 化進神 → 必成
   ├─ 化退神 → 難成
   └─ 空破 → 不成

2. 看世爻
   ├─ 旺相有氣 → 吉
   ├─ 空破 → 凶
   └─ 被克 → 有阻

3. 看兄弟爻
   ├─ 持世動 → 難成
   ├─ 動克財 → 有競爭
   └─ 化退神 → 阻力減

4. 看官鬼爻
   ├─ 間爻動 → 有阻撓
   ├─ 化退神 → 阻力弱
   └─ 空破 → 無礙

5. 綜合判斷
```

### 女占婚姻

```
1. 看官鬼爻
   ├─ 持世旺相 → 可成
   ├─ 化進神 → 必成
   ├─ 化退神 → 難成
   └─ 空破 → 不成

2. 看世爻
   ├─ 旺相有氣 → 吉
   ├─ 空破 → 凶
   └─ 被克 → 有阻

3. 看子孫爻
   ├─ 持世動 → 難成
   ├─ 動克官 → 長輩反對
   └─ 化退神 → 阻力減

4. 看財爻
   ├─ 動生官 → 助成
   ├─ 空破 → 不利
   └─ 旺相 → 有助

5. 綜合判斷
```
