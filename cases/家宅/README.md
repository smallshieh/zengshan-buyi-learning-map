# 🏠 家宅類卦例

## 📚 分類說明

家宅類卦例涉及房屋、居所相關的占卜,包括:
- 家宅吉凶
- 買宅、遷居
- 修造、動土
- 宅中六親安危

## 🎯 用神取法

占家宅以**[[glossary/父母爻|父母爻]]**為用神,因父母爻代表庇護我身之物。

### 相關六親
- **父母爻**:房屋本身,宜旺相有氣
- **子孫爻**:福德之神,持世主安寧
- **妻財爻**:財源,旺相主財運
- **官鬼爻**:憂疑之神,動則主不安
- **兄弟爻**:阻隔之神,動則主破財

## 💡 學習要點

1. **用神選擇**:占家宅、屋舍、車船等庇護之物,皆以父母爻為用神
2. **六親配合**:不僅看父母爻,還要看世爻、財爻、子孫等配合
3. **吉凶判斷**:
   - 父母旺相、子孫持世、財爻有氣 → 吉宅
   - 父母休囚、官鬼發動、兄弟劫財 → 凶宅
4. **特殊情況**:
   - 占買宅:財爻為價錢,父母為房屋
   - 占遷居:世動為動,世靜為不動
   - 占修造:父母動為動工,化吉則吉

## 🎓 實戰技巧

### 判斷宅之吉凶
1. 看父母爻旺衰
2. 看世爻所臨六親
3. 看動爻生克關係
4. 看六親整體配合

### 判斷宅之方位
根據父母爻所臨地支:
- 寅卯木:東方
- 巳午火:南方
- 申酉金:西方
- 亥子水:北方
- 辰戌丑未土:中央或四隅
