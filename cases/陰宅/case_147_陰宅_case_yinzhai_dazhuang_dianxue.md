# 陰宅｜點穴卦例：大壯

- **年月**：未月乙巳日
- **問事**：占穴（陰宅點穴）
- **卦象**：大壯卦

```
兄弟戌土″                兄弟戌土″  
子孫申金″                子孫申金″  
父母午火′世              父母午火′世
兄弟辰土′                兄弟辰土′  
官鬼寅木′                官鬼寅木′  
妻財子水′應              妻財子水′應
```

- **斷語**：世在四爻，穴在中段；午火持世，宜尋草木枯焦、紅色野花處。以紅原錢驗之，得錢處即穴，掘之土穴，周圍皆石塊。
- **應驗**：次年開府，兩公郎五年內俱登甲榜。
