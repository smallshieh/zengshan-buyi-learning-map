## 💡 核心學習要點

### 1. 用神判斷

#### 占壽元
- **用神**：世爻（代表自己）
- **元神**：生世之爻
- **忌神**：官鬼爻（克世）

### 2. 長壽判斷

✅ **長壽之象**：
- 世爻旺相
- 世臨日月
- 日月生扶世爻
- 動爻生扶世爻
- 動而化生化旺

❌ **短壽之象**：
- 世爻休囚
- 世爻空破
- 世爻化絕化墓
- 官鬼動克世
- 世爻化回頭克

### 3. 應期判斷

#### 凶兆應期
```
世爻休囚 → 沖克之年
世動化退 → 退神入墓之年
世化回頭克 → 變爻值年
世化絕墓 → 絕墓之年
多鬼入墓 → 鬼入墓之年
```

#### 元神忌神
```
元神動 → 不宜動（動則有限期）
忌神動 → 逢合逢值之年
```

### 4. 特殊理論

#### 不執古法
- 古法：初爻管五年，二爻管五年
- 野鶴：試四十餘載無驗，不以為法
- 正確：以世爻衰旺斷長短，不定具體年限

#### 多占之法
- 一卦難定，可過二十年再占
- 前後卦合看，判斷應期

## 🎯 實戰判斷流程

### 占壽元

```
1. 看世爻旺衰
   ├─ 旺相 → 長壽
   ├─ 休囚 → 防沖克之年
   └─ 空破 → 凶兆

2. 看日月關係
   ├─ 日月生扶 → 吉
   ├─ 日月克沖 → 凶
   └─ 日月合世 → 吉

3. 看動爻變化
   ├─ 化生化旺 → 吉
   ├─ 化絕化墓 → 凶
   └─ 化回頭克 → 凶

4. 看官鬼爻
   ├─ 鬼動克世 → 凶
   ├─ 多鬼搖發 → 入墓之年
   └─ 鬼空破 → 無礙

5. 判斷應期
   └─ 不定具體年限，只斷長短
```

## ⚠️ 注意事項

1. **只斷長短**：不定具體年限，只斷長壽或短壽
2. **元神宜靜**：元神動則有限期
3. **忌神宜靜**：忌神動則應期近
4. **多占合斷**：一卦難定，可過若干年再占

---

## 💡 經典理論

> **世爻旺相，永享長年，身位休囚，須防夭折。**  
> 凡占壽元，獨以世爻為根本，世爻或旺相、或臨日月、或得日月生扶及動爻生扶、或動而化生化旺化回頭生者，乃大壽之征也。

> **元神宜於安靜。**  
> 占壽，世為根本，元神為滋生之物，宜旺而靜不宜動搖，其故何也？占他事宜元神動者，動則而有力也，占壽元不宜發動，動則已有限期矣。

> **忌神最怕動搖。**  
> 忌神不動，自是平安，動則已有限期矣。非應逢合之年，必應逢值之歲。
