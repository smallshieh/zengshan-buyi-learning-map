# 📝 占驗詳情

> **原文**:卯月丁巳日占產期得離為火變火山旅卦

## 卦象

```
離為火 → 火山旅

兄弟巳火′世              兄弟巳火′  
子孫未土″                子孫未土″  
妻財酉金′                妻財酉金′應
官鬼亥水′應  動          妻財申金′  
子孫丑土″                兄弟午火″  
父母卯木′                子孫辰土″世
```

## 🔍 斷語邏輯

占生產以[[子孫爻]]為用神,因子孫代表子女。此卦世爻巳火臨[[兄弟爻]],非子孫,但卦中未土、丑土兩重子孫,當擇其旺者為用。

卯月木旺土相,未土子孫得月令之氣,又臨二爻,為用神。應爻亥水[[官鬼]]發動,官鬼為產母之憂疑,動則主臨產。亥水化申金[[妻財]],財為產母之身,化財則母安。

關鍵在於此卦為[[六沖卦]],占產期逢六沖,主速產易生。未土子孫與丑土相沖,沖則速也。

## ✅ 結果

斷曰:六沖卦主速產,官鬼動主臨產之期已近。未土子孫旺相,丑日沖之,當於丑日或丑時生產,母子平安。

果驗:次日戊午日丑時臨盆,順產得一子,母子均安。應丑時者,丑土沖未土子孫之時也。

## 💡 理論要點

1. **子孫為胎**:占生產以子孫爻為用神,代表胎兒
2. **[[六沖]]主速**:占產期逢六沖卦,主速產易生
3. **官鬼動**:官鬼為憂疑,動則主臨產之期
4. **應期判斷**:子孫逢沖之日時,即為生產之期
