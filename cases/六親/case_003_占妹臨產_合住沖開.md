# 📝 占驗詳情

> **原文**：午月戊辰日占妹臨產吉凶得火地晉

## 卦象


## 🔍 斷語邏輯

酉金[[兄弟爻]]為[[用神]]，月令克之，日建生之，許之無礙，卯日卯時必生。

## ✅ 結果

果于次日卯時生，母子平安。

## 💡 理論要點

1. **月克日生相敵**：午月克酉金，辰日生酉金，可以相敵
2. **[[合住]]**：酉金與辰日相合
3. **應期判斷**：卯日卯時，沖開合局

### 黃金策曰

> 若逢合住，必待衝開。

此月克而日生，無增克制或幫扶也。
