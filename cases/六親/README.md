## 💡 核心學習要點

### 1. 用神判斷

| 占何人 | 用神 | 元神 | 忌神 |
|--------|------|------|------|
| 父母 | 父母爻 | 官鬼爻 | 子孫爻 |
| 兄弟 | 兄弟爻 | 父母爻 | 官鬼爻 |
| 子女 | 子孫爻 | 兄弟爻 | 官鬼爻 |
| 妻子 | 妻財爻 | 子孫爻 | 兄弟爻 |
| 丈夫 | 官鬼爻 | 妻財爻 | 子孫爻 |

### 2. 吉凶判斷

✅ **吉象**：
- 用神旺相
- 用神化進神
- 元神動而生用神
- 忌神空破
- 忌神化退神

❌ **凶象**：
- 用神空破
- 用神化絕化墓
- 用神化回頭克
- 忌神動而克用神
- 元神空破

### 3. 疾病判斷

#### 占六親病
```
用神旺相 + 忌神空破 = 病愈
用神休囚 + 忌神發動 = 病重
用神化絕 + 鬼動克 = 病危
```

#### 特殊情況
- **三卦雷同**：連占三次皆同卦，凶兆更明
- **多占之法**：一卦難定，可再占合斷

## 🎯 實戰判斷流程

### 占六親病

```
1. 確定用神
   └─ 占何人？對應何爻？

2. 看用神狀態
   ├─ 旺相 → 吉
   ├─ 休囚 → 凶
   ├─ 空破 → 凶
   └─ 化絕墓 → 危

3. 看忌神狀態
   ├─ 空破 → 吉
   ├─ 化退神 → 吉
   ├─ 發動 → 凶
   └─ 化進神 → 凶

4. 看元神狀態
   ├─ 動而生用神 → 吉
   ├─ 空破 → 不吉
   └─ 被克 → 凶

5. 綜合判斷
```
