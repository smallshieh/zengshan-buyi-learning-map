# 陰宅｜父外任平安｜姤之恒

- **時間**：申月癸巳日
- **問事**：父外任平安
- **卦象**：姤之恒（伏吟）
- **斷語**：巳火日辰生父母，當許在任平安，獨憂卦得伏吟，乃是不祥之象，任上必有事故，不得意以呻吟也。日建生父，他事無礙。伏吟欲歸而不能，辰年可歸。後知苗獨作亂，地方不祥，驚險異常；寅年占卦，辰年缺歸，午年又官者，占時遇巳日扶起午火之官，當日亥水克之而不盡，今午火官星值年依然旺矣。
