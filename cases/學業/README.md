# 📚 學業類卦例

## 📚 分類說明

學業類卦例涉及求學、教育相關的占卜,包括:
- 延師(聘請老師)
- 學業成否
- 考試功名
- 師生關係

## 🎯 用神取法

### 占延師
以**[[glossary/父母爻|父母爻]]**為用神,因師長如父母也。

### 占學業
- **官鬼爻**:功名、學位
- **父母爻**:師長、學問
- **子孫爻**:不利功名(制官)

- **卦象**:山風蠱(六爻安靜)
- **要點**:父母旺相、官鬼持世、師生相宜
- **結果**:延師成功,學業大進

## 💡 學習要點

1. **用神選擇**:
   - 占延師 → 父母爻
   - 占學業 → 官鬼爻
   - 占考試 → 官鬼爻
2. **世應關係**:
   - 世為學生,應為老師
   - 應生世 → 師愛徒
   - 世生應 → 徒敬師
3. **吉凶判斷**:
   - 父母旺相 → 師有才學
   - 官鬼旺相 → 學業有成
   - 子孫發動 → 不利功名
4. **特殊情況**:
   - 占童試 → 看官鬼
   - 占延師 → 看父母
   - 占師生關係 → 看世應

## 🎓 實戰技巧

### 判斷師之優劣
1. 父母旺相 → 有才學
2. 官鬼有氣 → 有威嚴
3. 應爻生世 → 樂教導
4. 六爻安靜 → 師生和

### 判斷學業成否
1. 官鬼旺相 → 可成
2. 子孫發動 → 難成
3. 父母有氣 → 有助
4. 世爻旺相 → 自身努力
