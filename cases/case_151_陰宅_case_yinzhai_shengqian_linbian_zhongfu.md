# 陰宅｜升遷｜臨變中孚

- **時間**：卯月己亥日
- **問事**：升遷
- **卦象**：臨變中孚（外卦反伏）
- **斷語**：世臨卯木月建之官，又長生於亥日，世與官星同臨旺地，許之即升，果于本月聞報，由江西升任山東，未及一載複任江西。應本月升者卯官而值月令；升山東者官臨卯木；複任江西者，外卦反伏，去之而複反也。
