# 📝 占驗詳情

> **原文**：一日到一宅上見醫者滿座。卯月丙辰日占父病得地雷複卦

## 卦象

### 第一卦：地雷復

```
地雷復（靜卦）

子孫酉金″ 
妻財亥水″ 
兄弟丑土″應
兄弟辰土″  
官鬼寅木″  
妻財子水′世
```

父母用神不現，明知巳火父母伏于二爻寅木之下，旺木以生巳火，飛來生伏，必愈之症，不以此斷，令再卜之。

## 🔍 斷語邏輯

餘疑曰：兩卦俱當即愈，如何此卦亥水回頭克父母，忽而悟曰：是也，前兩卦巳火旺不受傷克，乃應目前之愈，此卦亥水克父，今冬難延。

即到前令自占，此卦竟於前卦相同，再請夫人卜之，又得此卦。

余連見此卦，毛骨松然，有神乎？無神乎？

占父亥水回頭克父，自占病亥水克世，妻占夫亥水克夫，三卦雷同，同如一手推出，今冬之危，扁鵲亦難為矣。

餘且不言，只以前卦斷目前之愈，問請醫曰：卦中不得見，列位各見何如？

曰：病勢甚險，遂公議一方，對症便好，不然只看陰功德行耳。

內有一位冷笑不言，餘請問姓，答曰，姓壽，因私問曰：此公之祥，何如？

彼曰：我可治之，不服我藥，奈何。

餘暗囑其子曰：太翁之病，須用姓壽者治之。

次早壽姓來謝，餘問將來何如？

壽曰：目下不妨，今冬難保。

餘曰：公言與數相合，真神醫也。

## ✅ 結果

- 目下即愈（前兩卦應驗）
- 果絕于亥月（第三卦應驗）

## 💡 理論要點

1. **多占之法**：用神不現，不取伏神，再占兩卦合斷
2. **三卦合斷**：
   - 第一卦：伏神飛來生伏（不取）
   - 第二卦：用神明現，旺相即愈
   - 第三卦：回頭克父，今冬難延
3. **三卦雷同**：父占、自占、妻占皆得同卦，神機莫測
4. **近遠分判**：前兩卦應目前之愈，第三卦應將來之死

### 理論要點

> 余之不取伏神，多占幾卦以決禍福而更穩。即知目前之生，又知將來之死，非多占之力耶？
