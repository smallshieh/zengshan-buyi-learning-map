## 🔑 核心學習點

### 1. 用神判斷
- **自占病**：以世爻為用神
- **占他人病**：以六親對應爻為用神
- **用神旺衰**：決定病情輕重

### 2. 生死判斷

#### 凶象
- **用神無根**：月破日克，雖有生扶生之不起
- **回頭克**：世爻化回頭克，自身受損
- **元神被克**：元神受制，用神失去生扶

#### 吉象
- **克處逢生**：雖被克制，得生扶而救
- **元神有力**：元神旺相生用神
- **暗動救援**：日辰沖動元神暗動生扶

### 3. 應期判斷

#### 病癒應期
- **生扶之日**：元神值日，生扶用神
- **沖去忌神**：沖去克制用神之爻
- **出空之日**：用神出空得力

#### 病危應期
- **忌神當令**：忌神值月日而克用神
- **元神受克**：元神被沖克之日
- **用神入墓**：用神入墓之日

### 4. 特殊理論

#### 用神無根
> 用神月破日克，值月破而被克，雖有生扶生之不起，如樹無根，寒谷不回春也。

#### 六神不主生死
> 虎興而遇吉神不害其吉，龍動而逢凶曜難掩其凶。

斷卦以五行生克為主，六神為輔。

#### 暗動福來不知
> 占以暗動福來而不知，禍來而不覺。

暗動之應較為隱蔽，但作用力不減。
