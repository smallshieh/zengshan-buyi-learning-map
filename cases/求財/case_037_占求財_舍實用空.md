# 📝 占驗詳情

> **原文**：未月庚子日占求財得風天小畜

## 卦象

```

兄弟卯木′                兄弟卯木′

妻財未土″應              妻財未土″應
妻財辰土′                妻財辰土′  
兄弟寅木′                兄弟寅木′  
父母子水′世              父母子水′世
```

## 🔍 斷語邏輯

應臨月建之財以克世，許之必得。

彼問何時到手，余以次日辛醜衝動未財日得，卻得於辰土出空之日。

此乃舍其不空而用旬空。

## ✅ 結果

得於辰土出空之日。

## 💡 理論要點

1. **用神兩現**：卦中兩爻妻財（未土、辰土）
2. **舍不空用旬空**：未土不空，辰土旬空，反應辰土
3. **出空之日**：辰日辰土出空而得財
4. **古法不驗**：古法舍休囚用旺相，此卦反之

### 理論要點

> 用神兩現，舍其休囚，用其旺相，舍其靜爻，而用動爻，舍其月破，而用不破，舍其旬空，用其不空。此古法也。得其驗者，應乎旬空月破，舍其不空，而用旬空，舍其不破，而用月破。
