## 💡 核心學習要點

### 1. 三爻並重

占終身財福，必須同時看三個爻：

| 爻位 | 名稱 | 作用 | 重要性 |
|------|------|------|--------|
| **世爻** | 自身 | 代表本人身體、壽命 | 根本 |
| **財爻** | 財帛 | 代表財富、收入 | 主體 |
| **子孫爻** | 福德 | 代表財源、福祿 | 源頭 |

### 2. 吉凶判斷

#### 🌟 最佳組合
```
世爻旺相 + 財爻旺相 + 子孫旺相
= 家殷戶足、福祿綿長
```

#### ⚠️ 偏重組合

| 組合 | 結果 | 說明 |
|------|------|------|
| 財世旺 + 福不旺 | 先富後貧 | 水無源流，終須枯涸 |
| 世福旺 + 財不旺 | 無財享福 | 坐享其成，不知生發 |
| 財福旺 + 世不旺 | 富屋貧人 | 雖有萬金，難買長生 |

#### ❌ 最差組合
```
世爻失陷 + 財爻失陷 + 子孫失陷
= 無衣乏食之人
```

### 3. 旺衰標準

#### 旺相之象
- 臨日月
- 得日月生扶
- 動而化吉
- 長生帝旺

#### 失陷之象
- 空破墓絕
- 動而變凶
- 被日月克
- 休囚無氣

### 4. 特殊情況

#### 世爻空破
- **動空**：沖空則實，實空之年勃發
- **靜空**：終身無成
- **旺空**：帶旺匪空，仍可成家
- **弱空**：一生貧困

#### 財爻化空破
- **動空**：實空之年得財
- **動破**：實破之年得財
- **靜空破**：難以聚財

## 🎯 實戰判斷流程

### 占終身財福

```
1. 看世爻
   ├─ 旺相有氣 → 身強
   ├─ 休囚無氣 → 身弱
   ├─ 空破墓絕 → 凶兆
   └─ 動而化吉 → 可成

2. 看財爻
   ├─ 持世旺相 → 富足
   ├─ 動而生世 → 得財
   ├─ 空破墓絕 → 貧寒
   └─ 化進化退 → 看趨勢

3. 看子孫爻
   ├─ 旺相發動 → 財源廣
   ├─ 休囚空破 → 財源斷
   ├─ 化進神 → 福祿增
   └─ 化絕墓 → 福祿減

4. 綜合判斷
   ├─ 三者皆旺 → 大富
   ├─ 兩旺一弱 → 小成
   ├─ 一旺兩弱 → 貧寒
   └─ 三者皆弱 → 赤貧
```

## 💡 經典理論

> **財福旺相，鐘鳴鼎食之家。**  
> 世爻旺相無傷，既富且壽。世爻財爻子孫爻三者無一失陷，定是家殷戶足、福祿綿長。

> **財世休囚，龜釜生塵之宅。**  
> 世爻財爻福爻三者無氣、或空破墓絕、或動而變凶，乃無衣乏食之人也。

> **水要看源，木要尋根。**  
> 用神為事之體，元神為事之本。用神雖旺，元神若被傷克者，即如水無泉源，木無根蒂。
