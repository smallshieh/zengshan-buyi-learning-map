## 💡 核心學習要點

### 1. 用神判斷
- **官鬼爻**為主要用神（代表功名）
- **父母爻**為元神（代表文章、文書）
- **子孫爻**為忌神（克官鬼）

### 2. 成名條件
✅ **吉象**：
- 官星持世旺相
- 父動生官
- 官化進神
- 日月生扶官星
- 財動生官（納栗）

❌ **凶象**：
- 官星空破
- 子孫持世發動
- 官化退神
- 世化回頭克

### 3. 成名途徑
1. **科舉正途**：父動官旺
2. **納栗援例**：財動生官
3. **立功建業**：白虎金官（武途）

### 4. 應期判斷
- 實破之年：月破動爻填實之年
- 實空之年：旬空爻填實之年
- 衝開墓庫：墓爻逢沖之年月
- 化進神：進神得令之時
