# 📝 占驗詳情

> **原文**：戌月己巳日余占南行得大有之大壯

## 卦象

### 第一次：戌月己巳日

```
火天大有 → 雷天大壯

官鬼巳火′應  動          父母戌土″  
父母未土″                兄弟申金″  
兄弟酉金′                官鬼午火′世
父母辰土′世              父母辰土′  
妻財寅木′                妻財寅木′  
子孫子水′                子孫子水′應
```

余以應爻為官星動而生世，官貴相生者，雖則可喜，但嫌巳火墓於戌月而又化墓，生我之官爻即又入墓耶，兼且又是六沖，疑為不吉。

### 第二次：亥月甲辰日

```
火天大有 → 山天大畜

官鬼巳火′應              妻財寅木′  
父母未土″                子孫子水″應
兄弟酉金′    動          父母戌土″  
父母辰土′世              父母辰土′  
妻財寅木′                妻財寅木′世
子孫子水′                子孫子水′  
```

餘疑又得此卦，巳火雖則生世又為月破，破而不能生我，這何益。況間爻酉金兄動乃阻隔耗財之神，疑之不行。

### 第三次：甲寅日

```
火天大有 → 雷天大壯

官鬼巳火′應  動          父母戌土″  
父母未土″                兄弟申金″  
兄弟酉金′                官鬼午火′世
父母辰土′世              父母辰土′  
妻財寅木′                妻財寅木′  
子孫子水′                子孫子水′應
```

連得三次，餘始悟，曰：此行到明年三四月必定如心。

巳火之官動而生世，目下雖破入墓，明年三月衝開戌墓，巳火得出，至巳月得令，當時不破不空，旺相而生世矣。

## ✅ 結果

即于巳日起程，行至中途遇舊識，母死又被回祿，同行，行不數日，又遇知己，道及某處有奇門一部，乃系抄本，命餘同去買之，同道兩日，余蔔一卦，又得大有之大壯，余對友曰：此卦得之四次，非叫我買書，乃促我行也。竟行，後至地頭，已二月矣。

## 💡 理論要點

1. **多占之法**：連占三次，皆得同卦
2. **化墓衝開**：巳火化墓，辰月衝開戌墓
3. **月破實破**：巳火月破，巳月實破而旺
4. **神機莫測**：四次得同卦，神促其行
5. **應期判斷**：三四月衝開墓庫，巳月得令
