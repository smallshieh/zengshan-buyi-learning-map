# 📝 占驗詳情

> **原文**：卯月辛巳日來人不言所事。占得巽卦變乾卦

## 卦象

```
巽為風 → 乾為天

巽木化乾金，木被金克
```

## 🔍 斷語邏輯

問之所占何事？

伊曰：代卜長輩功名。

餘曰：功名須要親占，代占難取用神，從不敢斷，幸此卦而易見，巽木化乾金，即為[[化克|化回頭克]]，為絕卦也，不必問名，壽命不長。

## ✅ 結果

果於削職後七月而終。

## 💡 理論要點

1. **[[化克]]**：巽木化乾金，金回頭克木
2. **不看用神**：卦變化克，不論用神衰旺皆以凶推
3. **絕卦**：木被金克為絕卦
4. **代占亦驗**：雖為代占，卦變凶兆明顯

### 理論要點

> 卦變者：變生、變克、變墓、變絕、比和。予得驗者，凡遇卦化克者，不論用神之衰旺皆以凶推。
