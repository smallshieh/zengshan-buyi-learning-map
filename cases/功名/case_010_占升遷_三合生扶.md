# 📝 占驗詳情

> **原文**：寅月丙申日占官之升遷得艮卦變山雷頤

## 卦象

```
艮為山 → 山雷頤

官鬼寅木′世              官鬼寅木′  
妻財子水″                妻財子水″  
兄弟戌土″                兄弟戌土″世
子孫申金′應  動          兄弟辰土″  
父母午火″                官鬼寅木″  
兄弟辰土″    動          妻財子水′應
```

## 🔍 斷語邏輯

寅木[[官鬼爻|官星]]持世臨[[月建]]而旺相，雖被申日沖克，喜申子辰[[glossary/三合局|三合水局|合成水局]]，以生官，不獨無害，定然三月高遷。

## ✅ 結果

果于三月升任雲南太守。

## 💡 理論要點

1. **世臨月建**：寅木官星持世，臨寅月月建
2. **日沖不散**：申日沖寅木，但因有生扶不散
3. **[[glossary/三合局|三合水局]]**：申子辰合成水局生寅木官星
4. **[[theory/13_克處逢生章|克處逢生]]**：雖被申日沖克，得水局生扶反為吉

### 應期分析
- **應三月者**：辰土出空之月
- **應雲南者**：世與官星皆在六爻（上爻主遠方）
