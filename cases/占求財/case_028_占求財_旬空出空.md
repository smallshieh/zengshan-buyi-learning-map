# 📝 占驗詳情

> **原文**：寅月庚戌日占求財得火天大有

## 卦象

```
火天大有

官鬼巳火′應         
父母未土″           
兄弟酉金′           
父母辰土′世         
妻財寅木′           
子孫子水′           
```

## 🔍 斷語邏輯

斷曰：寅木[[妻財爻|財爻]]為[[用神]]，財爻克世，此財必得，但目下尚空，要到甲寅日出空可得。

## ✅ 結果

果於甲寅得之。

## 💡 理論要點

1. **財爻克世反為得財**：財爻克世，反主得財之象
2. **[[旬空]]不為真空**：寅木財爻雖值旬空，但臨月建
3. **出空之日應驗**：甲寅日，寅木出空且填實
4. **逢空不空之辨**：古法謂「逢空不空」，野鶴老人認為在旬內畢竟為空

### 野鶴老人評論

> 若以逢空不空非也。若在旬內還作空亡，待出旬不為空也。
