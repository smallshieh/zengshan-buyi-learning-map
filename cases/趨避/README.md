## 📚 案例列表

### ✅ 成功避凶

#### [[case_047_占索債_避凶得救]]
- **日期**：卯月丙寅日
- **問題**：占索債得否
- **關鍵因素**：兄動化進克世、辰時出空
- **結果**：留過辰時，避免船覆江中 🌟
- **學習點**：世爻出空被克之時，避之則吉

**斷語精華**：
> 世爻落空，辰時出空，被日月動爻之克，必有危亡之禍。留之早膳，過此時辰去亦不遲。

### ❌ 未能避凶

#### [[case_048_占夢_未避木害]]
- **日期**：醜月戊子日
- **問題**：占夢（夢一身之血入河洗之）
- **關鍵因素**：元武木動克世、巽宮少女
- **結果**：未聽勸告，宿妾房遭劫傷
- **學習點**：元武木動克世，應避出外、勿貪少女

**避凶建議**：
- 世與木爻皆在內卦，出外可避
- 巽為少女，勿貪幼婦
- 交春可以避之

**實際結果**：
> 不意果於正月亥日宿于妾房，被賊明火入室，席捲一空，身受木器所傷。

#### [[case_049_占夢_世變鬼死]]
- **日期**：午月丁亥日
- **問題**：占夢前夫叫去
- **關鍵因素**：世變鬼、重土克世
- **結果**：臘月而死
- **學習點**：世爻變鬼，重重克制，難以避免

### 🛡️ 子孫護身

#### [[case_050_占夢_子持無災]]
- **日期**：戌月戊申日
- **問題**：占夢母叫去
- **關鍵因素**：子孫持世
- **結果**：竟無凶災
- **學習點**：凶夢相同，生死各別，子孫持世化凶為吉

**對比分析**：
> 凶夢相同，生死各別。前卦世爻變鬼，死于臘月。後卦子孫持世，竟無凶災。

## 💡 核心學習要點

### 1. 趨避原則

**聖人作易，原令人趨吉避凶**

✅ **可避之災**：
- 時辰可避：出空被克之時
- 方位可避：內卦在家，外卦出外
- 人事可避：避開特定人物（如少女）
- 季節可避：交春、交冬等

❌ **難避之災**：
- 世爻變鬼，重重克制
- 命中注定，無可避免

### 2. 吉凶判斷

#### 最凶之象
- **官鬼克世**：火鬼防火，木鬼防木害，水鬼防溺水
- **世爻變鬼**：動而化鬼，凶上加凶
- **元武動克**：盜賊劫財之災

#### 最吉之象
- **子孫持世**：飛殃橫禍化為塵
- **福神發動**：解災化煞

### 3. 五行災害對應

| 五行 | 災害類型 | 案例 |
|------|----------|------|
| 木鬼 | 木器傷害、樹木壓傷 | case_048 |
| 火鬼 | 火災、燒傷 | - |
| 土鬼 | 岩石、牆壁壓傷 | - |
| 金鬼 | 刀劍、金屬傷害 | - |
| 水鬼 | 溺水、水災 | case_047 |

### 4. 六神災害對應

| 六神 | 災害類型 |
|------|----------|
| 元武 | 盜賊、劫財 |
| 朱雀 | 官非、口舌 |
| 白虎 | 血光、刀兵 |
| 騰蛇 | 驚恐、怪異 |
| 勾陳 | 牢獄、田土 |
| 青龍 | 喜中生禍 |
