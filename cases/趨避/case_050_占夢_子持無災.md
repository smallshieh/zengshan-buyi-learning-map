# 📝 占驗詳情

> **原文**：戌月戊申日占夢母叫去已隨去矣得風山漸

## 卦象

```
風山漸（靜卦）

官鬼卯木′應              官鬼卯木′應
父母巳火′                父母巳火′  
兄弟未土″                兄弟未土″  
子孫申金′世              子孫申金′世
父母午火″                父母午火″  
兄弟辰土″                兄弟辰土″  
```

## 🔍 斷語邏輯

凶夢相同,生死各別。

前卦世爻變鬼,死于臘月。

後卦子孫持世,竟無凶災。

## ✅ 結果

竟無凶災。

## 💡 理論要點

1. **子孫持世**：申金子孫持世
2. **凶夢無凶**：雖夢母叫去,子孫持世無災
3. **生死各別**：同樣凶夢,因卦象不同,結果各異
4. **福德隨身**：子孫持世,飛殃橫禍化為塵

### 理論要點

> 凶夢相同,生死各別,前卦世爻變鬼,死于臘月,後卦子孫持世,竟無凶災。雖然子孫持世,難為吉兆,如逢月破,出月無憂。若臨旬空,許之出旬無憂。
