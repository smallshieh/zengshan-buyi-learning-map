# 📝 占驗詳情

> **原文**：辰月乙巳日占壽得中孚卦

## 第一次占卦（53歲）

### 卦象

```
風澤中孚（靜卦）

官鬼卯木′                官鬼卯木′  
父母巳火′                父母巳火′  
兄弟未土″世              兄弟未土″世
兄弟丑土″                兄弟丑土″  
官鬼卯木′                官鬼卯木′  
父母巳火′應              父母巳火′應
```

### 斷語

世臨未土,巳日生之,月建扶之,遂鬼孫期頤之祝,必享長年。

此卦象過二十年公再蔔之。

## 第二次占卦（73歲）

### 卦象

```
山澤損 → 地雷復

官鬼寅木′應  動          子孫酉金″  
妻財子水″                妻財亥水″  
兄弟戌土″                兄弟丑土″應
兄弟丑土″世              兄弟辰土″  
官鬼卯木′    動          官鬼寅木″  
父母巳火′                妻財子水′世
```

### 斷語

若是父爻鬼動,公之壽則不久矣。

今見多鬼搖發,反為不礙,今年太歲在子,遠享八年之福,至未年發是其時也。

未年鬼多入墓,又是世逢年破。

## ✅ 結果

果終於未年七月。

## 💡 理論要點

1. **世旺長壽**：世爻旺相,日月生扶,享長年
2. **二十年後再占**：不執古法初爻二爻定年限
3. **多鬼入墓**：多鬼搖發反不礙,入墓之年應期
4. **世逢年破**：未年世爻逢年破

### 理論要點

> 占以初爻管五年,二爻管五年,共作三十年。再占一卦,又作三十年,余試四十餘載,並無應驗,不以為法,止以世爻衰旺而斷知長短。
