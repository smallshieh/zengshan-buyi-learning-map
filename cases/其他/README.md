## 💡 核心學習要點

### 生產類
- **用神**：子孫爻
- **吉象**：子孫持世旺相
- **凶象**：子孫空破、官鬼動克

### 出行類
- **用神**：世爻（自己）、應爻（目的地）
- **吉象**：世應相生、財動生世
- **凶象**：世應相克、官鬼克世

### 風水類
- **用神**：父母爻（房屋、墳墓）
- **吉象**：父母持世旺相
- **凶象**：父母空破、被克

### 爭訟類
- **用神**：世爻（自己）、應爻（對方）
- **吉象**：世旺應衰、鬼化退神
- **凶象**：世衰應旺、鬼動克世
