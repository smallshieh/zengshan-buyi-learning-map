# 📝 占驗詳情

> **原文**:酉月甲寅日占延師得山風蠱卦

## 🔍 斷語邏輯

占延師以[[父母爻]]為用神,因師長如父母也。此卦世爻酉金臨[[官鬼]],非父母,但卦中子水、亥水兩重父母,當擇其旺者為用。

亥水父母臨五爻,得寅日木生,又得酉月金生,旺相有力。世爻酉金官鬼,官鬼為師之威嚴,持世示此師有德有威,可以為師。

應爻寅木[[兄弟]],兄弟為同輩,應爻為對方,寅木生亥水父母,示此師樂於教導。卦中六爻皆靜,無一爻克害父母,更驗此師可延。

## ✅ 結果

斷曰:父母旺相,世臨官鬼,此師有德有威,可以延請。六爻安靜,師生相安,必能教導有方。唯需以禮相待,不可輕慢。

果驗:延此師後,子弟學業大進,師生相得,三年後中舉。師之教導嚴而有方,深得家長敬重。

## 💡 理論要點

1. **父母為師**:占延師、占師長,皆以父母爻為用神
2. **官鬼持世**:官鬼為威嚴,持世示有威有德
3. **六爻安靜**:無克害,師生相安
4. **應爻生用**:應爻生父母,示對方樂於教導

## 💭 學習要點

1. **延師用神**:父母為師,官鬼為威,子孫為學
2. **世應關係**:
   - 世為自己(學生)
   - 應為對方(老師)
   - 應生世 → 師愛徒
   - 世生應 → 徒敬師
3. **吉凶判斷**:
   - 父母旺相 → 師有才學
   - 官鬼持世 → 師有威嚴
   - 六爻安靜 → 師生和睦
4. **特殊情況**:
   - 子孫動 → 不利學業(制官)
   - 兄弟動 → 多有阻隔
   - 財動克父 → 師生不睦
