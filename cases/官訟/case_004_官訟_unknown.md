# 📝 占驗詳情

> **原文**：同日妹占兄官事同此一案亦擬重罪得天地否變天水訟

## 卦象

```
天地否 → 天水訟

父母戌土′應              父母戌土′  
兄弟申金′                兄弟申金′  
官鬼午火′                官鬼午火′世
妻財卯木″世              官鬼午火″  
官鬼巳火″    動          父母辰土′  
父母未土″                妻財寅木″應
```

## 🔍 斷語邏輯

申金[[兄弟爻|兄爻]]為[[用神]]，巳火鬼動，刑克申金，重罪定矣。

幸喜辰日衝動戌土[[父母爻|父母]][[暗動]]生申，[[theory/13_克處逢生章|克處逢生]]，若有父母可以救之。

## ✅ 結果

後因父年八十，援例恩留免死。

## 💡 理論要點

1. **[[暗動]]救援**：辰日沖戌土，戌土暗動生申金
2. **[[theory/13_克處逢生章|克處逢生]]**：申金雖被巳火克，得戌土暗動生扶
3. **父母救子**：戌土父母生申金兄弟，應驗於實際父親救子
4. **援例免死**：因父年八十，援引律例恩留

## 對比分析

與[[cases/官訟/case_005_占父官事重罪|占父官事]]對比：
- 同一案件，不同占卜者
- 父占：用神無救，果至重刑
- 妹占：用神有救（官訟），援例免死
