# 📝 占驗詳情

> **原文**：午月丁未日占弟被訟吉凶得困卦變雷風恒卦

## 卦象

```
澤水困 → 雷風恒

父母未土″                父母戌土″應
兄弟酉金′    動          兄弟申金″  
子孫亥水′應              官鬼午火′  
官鬼午火″    動          兄弟酉金′世
父母辰土′                子孫亥水′  
妻財寅木″世              父母丑土″  
```

## 🔍 斷語邏輯

酉金[[兄弟爻|兄爻]]為[[用神]]，午月克之，未日生之，可以相敵，但不宜又動出午火相克，正所謂最怕他爻增克制。

彼問有害否？

餘曰：午火為[[月建]]動於卦中，謂之入卦者更見剛強，又謂之月建作[[忌神]]，為禍不淺，大凶之象。

又問凶在何時？

餘曰：酉金兄爻[[化退神]]，今歲辰年[[glossary/日辰|太歲]]相合，自是無妨恐，退至申年而無路矣。

## ✅ 結果

果于本年下獄至申年而重刑。

## 💡 理論要點

1. **月建入卦**：午火月建動於卦中，力量加倍
2. **他爻增克**：午月克酉金，午火又動克酉金，雙重克制
3. **[[化退神]]**：酉金化申金為退神，力量減弱
4. **太歲作用**：辰年太歲合酉金，暫時無妨
5. **應期判斷**：申年，退神之年而應凶

### 理論要點

> 日絕、日沖、日克，須察別位有生扶。化絕、化墓、化克，又怕他爻增制克。
