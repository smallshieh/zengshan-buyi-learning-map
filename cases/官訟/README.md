## 💡 核心學習要點

### 1. 用神判斷

#### 占官司訴訟
- **我方**：世爻
- **對方**：應爻
- **官府**：官鬼爻
- **解神**：子孫爻

#### 占官職
- **用神**：官鬼爻
- **原神**：財爻（生官）
- **忌神**：子孫爻（克官）

### 2. 勝負判斷

✅ **勝訴之象**：
- 子孫持世發動
- 世爻旺相，應爻休囚
- 官鬼化退神
- 子孫克官鬼

❌ **敗訴之象**：
- 官鬼動克世
- 世爻空破
- 應爻旺相，世爻休囚
- 子孫空破

### 3. 官職吉凶

✅ **吉象**：
- 官星持世旺相
- 財動生官
- 官化進神

❌ **凶象**：
- 世化回頭克
- 官化退神
- 子孫持世動（克官）
- 三爻皆凶

## 🎯 實戰判斷流程

### 占官司勝負

```
1. 看世應旺衰
   ├─ 世旺應衰 → 我勝
   └─ 世衰應旺 → 我敗

2. 看子孫爻
   ├─ 子孫持世 → 必勝
   ├─ 子孫動克鬼 → 勝
   └─ 子孫空破 → 不利

3. 看官鬼爻
   ├─ 鬼動克世 → 敗
   ├─ 鬼化退神 → 對方勢弱
   └─ 鬼空破 → 官府不理

4. 綜合判斷
```

### 占官職升降

```
1. 看官星狀態
   ├─ 官持世旺 → 有官
   ├─ 官化進神 → 升遷
   └─ 官化退神 → 降職

2. 看世爻狀態
   ├─ 世旺相 → 吉
   ├─ 世化回頭克 → 凶
   └─ 世空破 → 難成

3. 看子孫爻
   ├─ 子孫持世 → 無官
   ├─ 子孫動 → 克官
   └─ 子孫空 → 可成

4. 綜合判斷
```
