# 📝 占驗詳情

> **原文**：卯月丁巳日上下兩村因田水撕打得離之坤卦

## 卦象

```
離為火 → 坤為地

兄弟巳火′世  動          妻財酉金″世
子孫未土″                官鬼亥水″  
妻財酉金′    動          子孫丑土″  
官鬼亥水′應  動          父母卯木″應
子孫丑土″                兄弟巳火″  
父母卯木′    動          子孫未土″  
```

## 🔍 斷語邏輯

斷曰：內卦為我村，亥卯未[[glossary/三合局|三合木局|合成木局]]，外卦為人村，巳酉丑[[glossary/三合局|三合金局|合金局]]來克木，幸衰金不克旺木，不足畏也。況系[[六沖]]卦變六沖，有人解散必不成訟。

## ✅ 結果

後果勸散。

## 💡 理論要點

1. **內外卦分彼此**：內卦為我村，外卦為他村
2. **[[glossary/三合局|三合木局]]**：亥卯未合成木局（我方）
3. **[[glossary/三合局|三合金局]]**：巳酉丑合成金局（他方）
4. **旺衰判斷**：卯月木旺金衰，金不克木
5. **[[六沖變六沖]]**：必有人解散

### 或問

彼此之形，必以世應為主，如何不言世應？

餘曰：若無內外合局者，須看世應，今彼此兩村即內外兩卦人眾同心，彼此合局，神之如用靈驗顯然，故舍世應而不用也。
