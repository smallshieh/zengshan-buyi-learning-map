## 💡 核心學習要點

### 1. 用神判斷

#### 占行人歸期
- **用神**：父母爻（占父母）、兄弟爻（占兄弟）、子孫爻（占子女）等
- **元神**：生用神之爻
- **忌神**：克用神之爻

### 2. 歸期判斷

✅ **速歸之象**：
- 用神動而克世（行人回來之速）
- 用神化進神
- 用神旺相發動
- 日月生扶用神

🐌 **遲歸之象**：
- 用神靜而生世（行人歸來緩）
- 用神化退神
- 用神休囚
- 用神空破

❌ **不歸之象**：
- 用神伏藏被克
- 用神化絕
- 用神入墓
- 忌神動而克用神

### 3. 應期判斷

#### 用神動者
```
化進神 → 應進神得令之時
化退神 → 應退神入墓之時
動而逢沖 → 應沖之日
動而逢合 → 應合之日
```

#### 用神靜者
```
靜而旺相 → 應值之日或沖之日
靜而休囚 → 應生之日或旺之日
靜而空破 → 應出空實破之日
```

#### 特殊情況
```
月破化空 → 實空實破之日
伏藏被克 → 難有歸期
化絕入墓 → 凶多吉少
```

## 🎯 實戰判斷流程

### 占行人歸期

```
1. 確定用神
   └─ 占何人歸？對應何爻？

2. 看用神狀態
   ├─ 動還是靜？
   ├─ 旺還是衰？
   └─ 有無空破？

3. 看動爻變化
   ├─ 化進神 → 速歸
   ├─ 化退神 → 已歸或將歸
   ├─ 化絕墓 → 凶兆
   └─ 化回頭生克 → 看變爻

4. 判斷應期
   ├─ 動者逢值逢沖
   ├─ 靜者逢生逢旺
   └─ 空破者出空實破
```

## ⚠️ 注意事項

1. **用神選擇**：必須根據所占何人選擇對應六親爻
2. **動靜區別**：動克世速，靜生世遲
3. **化退神義**：化退神多為已歸或將歸之象
4. **實空實破**：空破逢填實之日為應期
