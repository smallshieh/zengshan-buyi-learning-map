# 📝 占驗詳情

> **原文**:午月壬申日占舟行安危得坎為水變水澤節卦

## 卦象


## 🔍 斷語邏輯

占出行、舟行以[[世爻]]為主,又以[[子孫爻]]為福德之神。此卦世爻子水臨[[兄弟爻]],雖非最吉,但子水得月令生扶,又臨日建,旺相有力。

關鍵在於四爻申金父母,被日辰壬申沖之,申金旺相,[[日辰]]沖之為[[暗動]]。申金暗動生子水世爻,又化進神,父母為舟船,暗動生世,示舟行雖有驚險,終得平安。

應爻午火[[妻財]]動化官鬼丑土,財化鬼本主破財驚恐,但丑土回頭克午火,反不能為害。

## ✅ 結果

斷曰:世爻旺相,父母暗動生世,舟行雖遇風浪,必得平安。唯需防午日或丑日,或有小驚,但無大礙。

果驗:行至中途遇大風,舟幾欲覆,幸得船工奮力,終於午時風停,化險為夷。眾人皆驚,唯占者心安,知神已先告也。

## 💡 理論要點

1. **[[暗動]]之理**:靜爻旺相,日辰沖之為暗動,有動之力
2. **父母為舟**:占舟行,父母爻代表船隻
3. **[[化進神]]**:申金化進,力量增強
4. **應期判斷**:午火財爻動,應於午時
