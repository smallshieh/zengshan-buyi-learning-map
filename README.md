## 📚 專案簡介

本專案將清代野鶴老人所著《增刪卜易》進行結構化整理，建立完整的學習導航系統，包含：

- ✅ **60+ 精選卦例**：從原文提取並轉換為 Markdown Frontmatter 格式
- ✅ **50 個核心術語**：建立專有名詞詞彙表，含詳細說明與交叉引用
- ✅ **16 大分類**：按問事類型分類（求財、功名、疾病、六親等）
- ✅ **200+ 交叉引用**：完整的雙向連結知識網絡

## 🎯 適用對象

- 六爻占卜初學者：系統化學習路徑
- 進階學習者：案例分析與術語查詢
- 研究者：結構化資料與交叉引用

## 🔍 卦例格式

每個卦例包含：

```yaml
---
tags: [增刪卜易/卦例, 類別/求財, 章節/元神忌神衰旺章]
date_lunar: 酉月辛亥日
question: 占求財
gua_original: 兌為澤
gua_changed: 雷水解
key_factors: [月破, 旬空, 化空, 動不為空]
result: 驗
---
```

- 📝 **占驗詳情**：原文引用與背景
- 🎲 **卦象**：本卦與變卦對照
- 🔍 **斷語邏輯**：推論過程（含術語連結）
- ✅ **結果**：驗證結果
- 💡 **理論要點**：關鍵學習點
- 🔗 **相關術語**：雙向連結

## 🌟 特色功能

### 1. 雙向連結系統

所有術語自動連結到詞彙表，詞彙表反向連結到相關卦例。

### 2. 對比學習

如 `case_005` 與 `case_006`：同一案件，不同占卜者，不同結果，深化理解。

### 3. 應期分析

詳細記錄應驗時間與理論依據，如：
- 酉時救活：酉金生亥水之時
- 巳亥日全生：巳日沖亥，亥日臨值

### 4. 野鶴老人評論

保留原文中野鶴老人的重要評論，如：
> 動爻逢空不為空，應於出空之日。
