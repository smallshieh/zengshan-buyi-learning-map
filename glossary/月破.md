# 月破

## 定義

**月破**指爻逢月建相沖，為無用之爻。

## 原文

> 月建沖爻即為月破，乃無用之爻也。

## 詳細說明

月建沖爻為月破，是六爻占卜中重要的衰弱標誌之一。月破之爻如同破損之物，難以發揮作用。

### 判斷方法

- 子月占卦，爻臨午火 → 午火月破
- 醜月占卦，爻臨未土 → 未土月破
- 寅月占卦，爻臨申金 → 申金月破
- 卯月占卦，爻臨酉金 → 酉金月破
- 辰月占卦，爻臨戌土 → 戌土月破
- 巳月占卦，爻臨亥水 → 亥水月破
- 午月占卦，爻臨子水 → 子水月破
- 未月占卦，爻臨丑土 → 丑土月破
- 申月占卦，爻臨寅木 → 寅木月破
- 酉月占卦，爻臨卯木 → 卯木月破
- 戌月占卦，爻臨辰土 → 辰土月破
- 亥月占卦，爻臨巳火 → 巳火月破

### 吉凶影響

- **用神月破**：諸占不利，難成之象
- **忌神月破**：反為吉兆，忌神無力
- **月破喜填**：逢合日、填實日可解
