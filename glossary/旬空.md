# 旬空

## 定義

**旬空**指爻值旬內所無之地支，為空亡。

## 原文

> 何謂旬空？如甲子至癸酉日為一旬，此十日之內，並無戌亥，以爻逢戌亥為空亡，又名旬空。

## 詳細說明

旬空是六爻占卜中的重要概念，指在十天干配十二地支的循環中，每一旬必有兩個地支不出現，爻臨此二支即為旬空。

### 六旬空亡表

| 旬名 | 空亡地支 |
|------|----------|
| 甲子旬 | 戌、亥 |
| 甲戌旬 | 申、酉 |
| 甲申旬 | 午、未 |
| 甲午旬 | 辰、巳 |
| 甲辰旬 | 寅、卯 |
| 甲寅旬 | 子、丑 |

### 空亡判斷

野鶴老人總結：
- **旺不為空**：爻旺相者不為空
- **動不為空**：動爻不為空，應於出空之日
- **有生扶不為空**：日月動爻生扶者不為空
- **真空為空**：休囚無氣者為真空

### 吉凶影響

- **用神旬空**：目下無用，待出空之日
- **忌神旬空**：反為吉兆
- **動而化空**：暫時不見，出空則應
- **旬空最愛填沖**：填實日、沖空日而應
