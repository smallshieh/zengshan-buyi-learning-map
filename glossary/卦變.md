# 卦變

## 定義

**卦變**指卦中有動爻時,動爻變化後形成新的卦象。原卦稱為「本卦」或「主卦」,變化後的卦稱為「變卦」或「之卦」。

## 基本概念

### 卦變的形成

1. **靜卦**:六爻皆靜,無卦變
2. **一爻動**:一個爻發動,形成一個變卦
3. **多爻動**:多個爻發動,形成一個變卦
4. **六爻皆動**:六爻全動,稱為「六爻亂動」

### 卦變的表示

```
本卦 → 變卦
震為雷 → 雷火豐
```

## 卦變的作用

### 1. 判斷吉凶

- **本卦**:主當前狀態
- **變卦**:主未來趨勢
- **動爻**:主變化過程

### 2. 應期判斷

- **動而逢合**:應於合日
- **動而逢沖**:應於沖日
- **化進化退**:應於進退之日

### 3. 生克關係

卦變後需要考慮:
- 動爻與變爻的生克關係(回頭生、回頭克)
- 變卦中各爻的旺衰
- 變卦的整體格局

## 卦變的類型

### 按變化程度分類

| 類型 | 定義 | 特點 |
|------|------|------|
| **一爻動** | 只有一個爻發動 | 變化單純,易於判斷 |
| **二爻動** | 兩個爻發動 | 需綜合判斷 |
| **三爻動** | 三個爻發動 | 變化複雜 |
| **四爻動** | 四個爻發動 | 以靜爻為主 |
| **五爻動** | 五個爻發動 | 以靜爻為主 |
| **六爻動** | 六爻全動 | 以變卦為主,或看世應 |

### 按卦象變化分類

| 類型 | 定義 | 例子 |
|------|------|------|
| **變六沖** | 變為六沖卦 | 占合宜散,占散宜合 |
| **變六合** | 變為六合卦 | 占合更吉,占散不利 |
| **變遊魂** | 變為遊魂卦 | 主不定、遊移 |
| **變歸魂** | 變為歸魂卦 | 主歸復、返回 |

## 實際應用

### 占斷原則

1. **動爻為主**:卦變以動爻為主導
2. **變爻為輔**:變爻影響動爻的吉凶
3. **本變合參**:本卦主現在,變卦主將來

### 特殊情況

- **化進神**:變卦更旺,主事情向好發展
- **化退神**:變卦更弱,主事情向壞發展
- **化絕化墓**:變入絕地或墓地,多主不吉
