# 出空

## 定義

**出空**指原本值旬空的爻,在特定的時間(日或時辰)脫離空亡狀態,開始發揮作用的過程。出空之日往往是事情應驗的關鍵時機。

## 出空時機

### 1. 時間出空

爻值旬空,待該地支出現時即為出空:

| 旬空地支 | 出空時間 |
|---------|---------|
| 甲子旬(戌亥空) | 戌日、亥日或戌時、亥時 |
| 甲戌旬(申酉空) | 申日、酉日或申時、酉時 |
| 甲申旬(午未空) | 午日、未日或午時、未時 |
| 甲午旬(辰巳空) | 辰日、巳日或辰時、巳時 |
| 甲辰旬(寅卯空) | 寅日、卯日或寅時、卯時 |
| 甲寅旬(子丑空) | 子日、丑日或子時、丑時 |

### 2. 動而出空

動爻逢空,待出空之日而應,稱為「動不為空」。

### 3. 沖空出空

空爻被日辰或動爻沖,也可視為出空。

## 實際應用

### 吉凶判斷

- **吉神出空**:吉神空而待出空,出空之日即為得吉之時
- **凶神出空**:凶神空而待出空,出空之日即為遇凶之時
- **用神出空**:用神空而旺相,出空之日事成

### 應期判斷

- **出空應期**:空爻出空之日,往往是事情應驗之期
- **動爻出空**:動爻化空,待出空之日而應
- **用神出空**:用神旺相而空,出空之日事成

## 經典案例

### 案例:占求財動不為空

> 酉月辛亥日占求財得兌為澤變雷水解卦

**卦象分析**:
- 巳火官鬼發動化寅木
- 寅木值旬空(化空)
- 巳火動爻不為空,待出空之日而應

**結果**:果於寅日早見木日得財

**理論要點**:
- 動不為空:巳火雖化空,但動爻不為空
- 出空應期:甲寅日,寅木出空
- 寅木生巳火,巳火生世
