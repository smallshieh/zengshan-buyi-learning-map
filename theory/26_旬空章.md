## 🎯 核心理論

### 旬空定義

**旬空**：旬內所無為旬空。十天干配十二地支，必有兩個地支在旬內不現，這兩個地支即為旬空。

## 💡 空亡判斷

### 動空與靜空

| 狀態 | 判斷 | 說明 |
|------|------|------|
| **動空** | 動則不空 | 動爻逢空，沖空則實 |
| **靜空** | 真空 | 靜爻逢空，出空則實 |

### 旺空與弱空

| 狀態 | 判斷 | 說明 |
|------|------|------|
| **旺空** | 帶旺匪空 | 旺相逢空，沖空則實 |
| **弱空** | 真空 | 休囚逢空，出空則實 |

## 🎯 實戰應用

### 占求財

```
財爻空亡：
- 動空 → 沖空則得
- 靜空 → 出空則得
- 旺空 → 帶旺匪空，仍可得
- 弱空 → 真空，難得
```

### 占功名

```
官爻空亡：
- 動空 → 沖空則中
- 靜空 → 出空則中
- 旺空 → 帶旺匪空，仍可中
- 弱空 → 真空，難中
```

### 占疾病

```
鬼爻空亡：
- 近病逢空即愈
- 久病逢空難愈
- 動空 → 沖空則凶
- 靜空 → 出空則凶
```

## ⚠️ 注意事項

### 1. 動不為空

動爻逢空，不作空論，沖空則實。

### 2. 旺空匪空

旺相逢空，帶旺匪空，沖空則實。

### 3. 多占之法

旬空難定，可多占合斷。

### 4. 出空應期

- **靜空**：出空之日填實
- **動空**：沖空之日填實
- **旺空**：沖空之日填實
