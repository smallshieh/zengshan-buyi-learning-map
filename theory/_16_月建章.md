# 月建章

> 本章探討月建在占卜中的作用與權重，月建是卦中權重最高的因素。

**章節位置**：卷二 第16章

## 🎯 核心理論

### 月建定義

**月建**：占卜當月的地支，對卦爻有最高生克權。

### 月建的地位

**月建如天**：
- 月建是卦中權重最高的因素
- 爻臨月建，力量最強
- 月建生扶，休囚亦旺
- 月建克制，旺相亦衰

---

## 💡 月建權重

### 權重順序

1. **月建** - 最高權重（如天）
2. **日辰** - 次高權重
3. **動爻** - 第三權重
4. **靜爻** - 最低權重

### 月建的作用

| 作用 | 說明 | 影響 |
|------|------|------|
| **生扶** | 月建生扶用神 | 休囚亦旺 |
| **克制** | 月建克制用神 | 旺相亦衰 |
| **沖破** | 月建沖爻 | 月破 |
| **合住** | 月建合爻 | 合住 |
| **臨月建** | 爻臨月建 | 力量最強 |

## 🎯 實戰應用

### 月建生扶

```
用神休囚 + 月建生扶 = 用神旺相
```

**效果**：休囚爻得月建生扶，如枯木逢春。

### 月建克制

```
用神旺相 + 月建克制 = 用神衰弱
```

**效果**：旺相爻被月建克制，力量大減。

### 月破判斷

```
爻被月建沖 = 月破
```

**相關**：[[glossary/月破|月破]]、[[theory/27_月破章|月破章]]

### 臨月建

```
爻臨月建 = 力量最強
```

**效果**：如君臨天下，不可撼動。

## ⚠️ 注意事項

### 1. 月建權重最高

月建如天，是卦中權重最高的因素。

### 2. 月建生扶最有力

月建生扶，休囚爻亦可旺相。

### 3. 月建克制最嚴重

月建克制，旺相爻亦會衰弱。

### 4. 月破需特別注意

月建沖爻為月破，詳見[[theory/27_月破章|月破章]]。
