## 🎯 核心理論

### 進神定義

**進神**：同類五行由衰變旺，表示事物向好的方向發展。

```
子 → 亥（水進神）
丑 → 子（土進神）
寅 → 卯（木進神）
卯 → 辰（木進神）
辰 → 巳（土進神）
巳 → 午（火進神）
午 → 未（火進神）
未 → 申（土進神）
申 → 酉（金進神）
酉 → 戌（金進神）
戌 → 亥（土進神）
亥 → 子（水進神）
```

### 退神定義

**退神**：同類五行由旺變衰，表示事物向壞的方向發展。

```
亥 → 子（水退神）
子 → 丑（土退神）
卯 → 寅（木退神）
辰 → 卯（木退神）
巳 → 辰（土退神）
午 → 巳（火退神）
未 → 午（火退神）
申 → 未（土退神）
酉 → 申（金退神）
戌 → 酉（金退神）
亥 → 戌（土退神）
子 → 亥（水退神）
```

## 💡 判斷方法

### 進神判斷

| 原爻 | 變爻 | 五行 | 判斷 |
|------|------|------|------|
| 申金 | 酉金 | 金 | 進神 ✅ |
| 寅木 | 卯木 | 木 | 進神 ✅ |
| 巳火 | 午火 | 火 | 進神 ✅ |
| 亥水 | 子水 | 水 | 進神 ✅ |

### 退神判斷

| 原爻 | 變爻 | 五行 | 判斷 |
|------|------|------|------|
| 酉金 | 申金 | 金 | 退神 ❌ |
| 卯木 | 寅木 | 木 | 退神 ❌ |
| 午火 | 巳火 | 火 | 退神 ❌ |
| 子水 | 亥水 | 水 | 退神 ❌ |

## ⚠️ 注意事項

### 1. 必須同類五行

進退神必須是**同類五行**的變化：
- ✅ 申金 → 酉金（金進神）
- ❌ 申金 → 午火（非進退神，是化克）

### 2. 旺衰方向

- **進神**：由衰變旺（如申→酉，酉為金之帝旺）
- **退神**：由旺變衰（如酉→申，申為金之長生但不如酉旺）

### 3. 結合其他因素

進退神需結合：
- 日月旺衰
- 空破狀態
- 回頭生克

### 4. 應期判斷

- **化進神**：應進神得令之時
- **化退神**：應退神入墓之時
